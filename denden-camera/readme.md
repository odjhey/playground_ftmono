# Denden Camera
## Steps
1. install @root `yarn` or `npm i`
1. go to server `cd server`
1. `yarn` or `npm i`
1. `yarn start` to start server
1. go to app `cd ../app`
1. install react-native cli `yarn` or `npm i`
1. `cd components`
1. install app `yarn` or `npm i`
1. run! `yarn android` or `npm run android`
